This Django project provides a RESTful API to fetch all users stored in a shared MySQL database (containerized using Docker).

- Clean, documented API using Django REST Framework

## Tech Stack

- Python 3.12
- Django 3.x
- Django REST Framework
- MySQL (Dockerized)
- Docker Compose
